Creating a function using the wizard

The easiest way to create a function is by using the built-in wizard. The wizard already contains various function types which can be customized as needed.

 

What is the purpose of functions

Functions are sets of points (x, y). They are mostly used as indicator diagrams, in other words, they represent data that cannot be computed. This kind of data is usually obtained by measurement.

In order to compute some of the engine's values, indicator diagrams are needed. The user has thus an option to create his own indicator diagrams.

 

To create an engine using a wizard:

·         Click File, New, Function (wizard)

·         Click Next

 

·         Different function types will appear

 

 

·         At this point, you can click Finish and the selected function type will be created. It is however possible to make many more customizations; click Next.

 

·         Depending on the type of the selected function, different Min/Max possibilities will appear; these serve as reference points for function creation.

·         In the next section, the number of starting points can be set. A base function will be then generated containing the selected number of points.

 

·         The last option is the possibility of altering the generated function

Function wizard showing the context menu for adding or removing points

 

·         The wizard procedure is now complete. By clicking Finish a new function will be generated.
