Creating an engine using the wizard

The easiest way to create an engine is by using the built-in wizard. The wizard already contains various engine layouts which can be customized as needed.

 

To create an engine using a wizard

·         Click File, New, Engine (wizard)

·         Click Next

·         Select the engine's cycle and click Next

 

·         Based on the selected cycle different layout possibilities will appear

 

 

·         At this point, you can click Finish and the selected engine layout will be created. It is however possible to make many more customizations; click Next.

 

·         In the Bore and stroke section the two fundamentals can be adjusted and the displacement varies accordingly. Fixing one or the other component and changing the displacement, will vary only the other one.

 

 

·         In the next section, you can vary the piston's mass

·         ...and the length of connecting rod

 

·         As connecting rods produce rotating and reciprocating forces, in addition to the mass, also the mass distribution is important. The wizard allows the distribution of the mass to be varied 100% towards rotating or 100% towards reciprocating. As a rule of thumb, the default is 2/3 reciprocating vs. 1/3 rotating.

 

  

·         The next section regards balancers masses and their rotation radiuses. If the balancer mass is chosen to be zero, the rotation radius will be disabled as it would be irrelevant.

 

·         The wizard procedure is now complete. By clicking Finish a new engine will be generated.
