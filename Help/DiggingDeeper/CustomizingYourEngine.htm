Customizing your engine

A wizard built engine can be customized further. A unique feature found in Engine designer is that the engine is comprised of "positioned cylinders", that is, separate one cylinder engine units. In this way any given engine can be represented and even different cycles can be put together.

 

The properties panel with the first cylinder's properties exposed

 

The properties panel always displays the engine's properties, divided by positioned cylinders. In this way each cylinder can be customized as needed.

 

Cylinder properties and their meaning

·         Position [integer, read only]: defines the cylinder's numerical position within the engine

·         Tilt [arc degrees]: defines the cylinder's tilt angle within the engine (measured from engine's vertical line, clockwise)

·         Offset [millimeters]: defines the cylinder's center phisical position within the engine (measured from engine's front plane)

·         Firing angle [arc degrees]:defines the cylinder's firing angle within the engine

·         Cycle: defines the cylinder's working cycle

·         Piston [the piston unit to be used with this cylinder]

o    Diameter [millimeters]: defines the piston's diameter (resulting in cylinder's bore)

o    Mass [grams]: defines the total mass of the piston (accessories included)

o    Skirt length [millimeters]: defines the total (max  measurable) length of the piston's skirt (from topmost point to lowest point)

o    Gudgeon pin distance from top of the piston [millimeters]: defines the distance of the center of the gudgeon pin from the topmost point of the piston 

·         Connecting rod [the connecting rod unit to be used with this cylinder]

o    Length [millimeters, read only]: indicates the total (max measurable) length of the connecting rod

o    Mass [grams, read only]: indicates the total mass of the connecting rod (all included)

o    Rotating mass [grams]: defines the rotating part of mass of the connecting rod (for dynamic equivalency)

o    Rotating mass distance from CG [millimeters]: defines the distance of the rotating part of mass from the connecting rod's center of gravity (for dynamic equivalency)

o    Reciprocating mass [grams]: defines the reciprocating part of mass of the connecting rod (for dynamic equivalency)

o    Reciprocating mass distance from CG [millimeters]: defines the distance of the reciprocating part of mass from the connecting rod's center of gravity (for dynamic equivalency)

·         Crank throw [the crank throw unit to be used with this cylinder]

o    Crank rotation radius [millimeters]: defines the crank throw's rotation radius (resulting in cylinder's stroke)

o    Balancer mass [grams]: defines the mass of the crank balancer

o    Balancer rotation radius [millimeters]: defines the rotation radius of the crank balancer

o    Balancer angle [arc degrees]: defines the angle of the balancer with respect to the crank throw

o    Crank pin width [millimeters]: defines the width of the crank pin

·         Bore [millimeters, read only]: indicates the cylinder's bore (based on piston's diameter)

·         Stroke [millimeters, read only]: indicates the cylinder's stroke (based on crank throw's crank radius)

·         Displacement [cubic centimeters, read only]: indicates the cylinder's total displacement (resulting from defined bore and stroke)

·         Stroke/Bore ratio [read only]: indicates the cylinder's stroke to bore ratio

·         Bore/Stroke ratio [read only]: indicates the cylinder's bore to stroke ratio

·         R/L ratio [read only]: indicates the crank throw's rotation radius vs. connecting rod length

·         L/R ratio [read only]: indicates the cylinder's connecting rod length vs. crank radius

·         Cylinder height [millimeters, read only]: indicates the required cylinder's height to accommodate the piston within its stroke

 

Engine properties and their meaning

·         Cylinders: Defines the cylinders and their positioning within the engine

·         Total displacement [cubic centimeters, read only]: Indicates the engine's total displacement

·         Crank throws [arc degrees, read only]: Indicates the engine's crank throws angles (resulting from defined cylinder setups) per array position, where the array position is the cylinder number (0 = first)

·         Firing order [integer, read only]: Indicates the engine's firing order, in cylinder positions

·         Firing angles [arc degrees, read only]: Indicates the engine's firing angles (resulting from defined cylinder setups), per array position, where the array position is the cylinder number (0 = first)

·         Number of cylinders [integer, read only]: Indicates the total number of cylinders in this engine (resulting from defined positioned cylinders)

·         Combined cycle duration [arc degrees, read only]: Indicates the duration of the engine's combined cycle (important when an engine has cylinder with different cycles)

·         Revolutions to complete combined cycle [integer, read only]: Indicates the number of revolutions needed to complete the engine's combined cycle
